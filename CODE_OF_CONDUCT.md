# Code of Conduct

`isaac-sim-rs` adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). The full text is available at that URL.

## Reporting

Contact: open a confidential GitHub issue or email the maintainer via the address on the GitHub profile. Reports are handled confidentially by Astro Robotics.
